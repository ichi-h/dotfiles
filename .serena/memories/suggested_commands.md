# Suggested Commands

## Development
- Build: Use nix commands within the flake directory
- Apply home-manager: `home-manager switch --flake .`

## Utilities
- `git` - Version control
- `curl` - HTTP requests
- Standard Linux tools: `ls`, `cd`, `grep`, `find`, etc.

## Notes
- This is a dotfiles repo, no build/test/lint commands for the repo itself
- Changes are applied via `home-manager switch` or NixOS rebuild
